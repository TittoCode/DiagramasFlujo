# DiagramasFlujo - Plataforma Educativa Interactiva

Una plataforma web interactiva diseñada para enseñar los fundamentos de la programación a través de diagramas de flujo. Este proyecto combina teoría clara, visualización de algoritmos y simuladores prácticos para facilitar el aprendizaje de lógica de programación.

![DiagramasFlujo Preview]( https://tittocode.github.io/DiagramasFlujo/ )

## Características Principales

*   **Diseño Educativo:** Interfaz limpia y profesional optimizada para el aprendizaje, con una paleta de colores suave que reduce la fatiga visual.
*   **Simuladores Interactivos:** Ejecuta algoritmos paso a paso (Área del Círculo, Par/Impar, Suma 1-100, Mayor de 3, Tablas de Multiplicar).
*   **Visualización de Flujo:** Los diagramas SVG se iluminan en tiempo real para mostrar el camino exacto que sigue la ejecución del código.
*   **Teoría Integrada:** Explicaciones claras sobre símbolos estándar (ANSI/ISO) y estructuras de control (Secuencial, Condicional, Repetitiva).
*   **Quiz de Evaluación:** Pon a prueba tus conocimientos con un cuestionario interactivo que ofrece retroalimentación inmediata.
*   **Totalmente Responsivo:** Funciona perfectamente en computadoras de escritorio, tabletas y dispositivos móviles.
*   **Modo Oscuro:** Soporte nativo y automático para temas claros y oscuros.

## Tecnologías Utilizadas

*   **HTML5 Semántico:** Estructura accesible y optimizada.
*   **Tailwind CSS:** Framework de utilidad para un diseño moderno, flexible y responsivo.
*   **JavaScript (ES6+):** Lógica interactiva pura, sin dependencias pesadas.
*   **SVG:** Gráficos vectoriales para diagramas nítidos y animables.

## Cómo Usar

No se requiere instalación ni servidor backend. Este es un proyecto estático puro.

1.  **Clonar:** Descarga el repositorio o clónalo con `git clone`.
2.  **Abrir:** Navega a la carpeta del proyecto.
3.  **Ejecutar:** Abre el archivo `index.html` en tu navegador web favorito (Chrome, Firefox, Edge, Safari).

## Contenido Educativo

El sitio cubre los siguientes temas clave:

1.  **Introducción:** ¿Qué son los diagramas de flujo y para qué sirven?
2.  **Simbología:** Terminal, Proceso, Decisión, Entrada, Salida, Conectores.
3.  **Estructuras de Control:**
    *   Secuenciales
    *   Condicionales
    *   Repetitivas (While, Do-While, For)
4.  **Ejemplos Prácticos:** Casos de uso reales con diagramas y código.
